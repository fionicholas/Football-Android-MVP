# kotlinMVP-football
Football Match App with Kotlin and Design Patern MVP

- List Match of League
- Standing League
- List Team of League
- Save favorite Match
- Save favorite Team
- Search Team or Match
- Load API with Retrofit
- Anko Sqlite
